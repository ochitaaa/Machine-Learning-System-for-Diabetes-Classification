# Machine-Learning-System-for-Diabetes-Classification
This repository contains documentation and supporting materials for building an end-to-end machine learning workflow, including model development, experiment tracking, monitoring, logging, and CI workflow. This project was developed as a module submission for the Asah by Dicoding program.

---

## Project Scope
This project focuses on:
- Building a machine learning classification model for diabetes prediction  
- Implementing experiment documentation and workflow automation  
- Monitoring and logging model performance  
- Applying CI workflow for machine learning projects  

---

## Repository Structure

- `Eksperimen_SML_Rosita-Angel-Cahyadi.txt`  
  Documentation of machine learning experiments and results.

- `Membangun_model/`  
  Materials and artifacts related to model development.

- `Monitoring dan Logging/`  
  Evidence and documentation for monitoring and logging implementation.

- `Workflow-CI.txt`  
  Documentation related to CI workflow for the machine learning system.
